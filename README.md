# Data_Quality_Checker
Python-based data quality checker that validates CSV datasets by identifying missing values, duplicates, and invalid data ranges. Designed to simulate real-world data validation workflows.
# Data Quality Checker (Python)

A simple Python project that checks data quality issues in CSV files such as:
- Missing values
- Duplicate records
- Invalid age ranges

## Technologies Used
- Python
- Pandas

## Features
- Reads CSV input data
- Generates a data quality report
- Identifies data issues for further analysis

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the script:
   python data_quality_checker.py

## Sample Output
- Missing value counts per column
- Number of duplicate rows
- Records with invalid age values
